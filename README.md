# Тестова работа выполненная для компании IT Craft

---

## Структура проекта

---

Пожалуйста, убедитесь, что файл релиза распакован в каталоге, доступном через Интернет. Вы увидите следующие файлы и каталоги

	src/		корневая папка проекта
	src/img		папка в которой находятся изображения
	src/sass	папка в которой находятся SASS-стили
	src/index.js	точка входа в webpack

## Требования

---

Минимальное требование для запуска проекта - наличие node.js v.8.9.2 и npm v.6.5.0

	node -v
	npm -v


## Установка

---

Клонировать проект VCS -> Checkout from version control -> GitHub

	git clone https://github.com/shishlachev-ilya/craft-test.git

Установить зависимости
	
	npm install

Запустить сборку для разработки

	npm run dev

Запустить сборку для продакшена

	npm run build
